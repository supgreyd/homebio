# HomeBio - WordPress Real Estate Project Plan

## ⚠️ Important Note: Hosting Consideration

**WordPress + Netlify Incompatibility:** WordPress is a PHP-based CMS requiring PHP and MySQL database. Netlify only hosts static sites and doesn't support PHP.

**Recommended Alternatives:**
1. **WordPress.com** - Managed WordPress hosting (easiest)
2. **Bluehost / SiteGround / Hostinger** - Traditional WordPress hosting (affordable)
3. **DigitalOcean / Vultr** - VPS with WordPress (more control)
4. **Cloudways** - Managed cloud WordPress hosting
5. **Local by Flywheel** - For development/testing

### Task 8.2: Redirect Rules
```php
// Redirect non-logged-in users trying to access cabinet
add_action('template_redirect', 'restrict_cabinet_access');
function restrict_cabinet_access() {
    if (is_page('user-cabinet') && !is_user_logged_in()) {
        wp_redirect(home_url('/login?redirect_to=' . urlencode($_SERVER['REQUEST_URI'])));
        exit;
    }
}

// Redirect after login
add_filter('login_redirect', 'custom_login_redirect', 10, 3);
function custom_login_redirect($redirect_to, $request, $user) {
    return home_url('/user-cabinet');
}
```

## Quick Start Commands

```bash
# Local development with WP-CLI
wp core install --url="homebio.local" --title="HomeBio" --admin_user="admin" --admin_email="admin@homebio.local"

# Create custom post type
wp scaffold post-type property --label="Properties" --textdomain="homebio"

# Install plugins
wp plugin install advanced-custom-fields --activate
wp plugin install ultimate-member --activate
wp plugin install polylang --activate

# Export/Import database
wp db export backup.sql
wp db import backup.sql
```
